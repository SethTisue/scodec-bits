  13. def hex(args: ByteVector*): ByteVector

    Converts this hexadecimal literal string to a ByteVector.

    Converts this hexadecimal literal string to a ByteVector. Whitespace characters are ignored.

    Named arguments are supported in the same manner as the standard s interpolator but they must be of type ByteVector.
